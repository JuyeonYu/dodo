## 소개 
두두는 둘이 쓰는 할 일 공유앱입니다.

아내와 함께 쓰려고 만들었습니다.

연인끼리 할일을 공유하는 것은 중요합니다.

퇴근길에 사다줄 물건을 부탁하기도 하고

출근길에 분리수거하는 것도 잊으면 안되죠.

또 상대방이 무슨일을 하는지 했는지 아는 것도 중요하죠.

같이 쓸 사람을 초대하고 서로에게 할 일을 만들어보세요!

### 다음 배포를 위해 열심히 개발중
- 상대방이 나에게 할일을 만들면 푸시로 알려드립니다!

---

#### 예정 기능
- 할일에 댓글을 달아 봅시다
- 할일에 카테고리 추가
- 달력으로 언제 뭘했는지 보기
- 이미지 추가
- 일정 추가
- 일정이 다가오면 푸시로 알려줌
